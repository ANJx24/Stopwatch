# ⏱️ Stopwatch

A modern stopwatch web application built with HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://anjx24.github.io/Stopwatch/

## ✨ Features

- ⏱️ Stopwatch with millisecond precision
- 📝 Lap recording
- 🗑️ Clear laps
- 🎤 Voice commands
- ⏳ Countdown timer
- 🍅 Pomodoro timer
- 📱 Responsive design
- 🌙 Theme support
